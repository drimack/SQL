## 🧩 What's is SQL?

SQL is acronym of Structured Query Language. Yes, it is a programming language :-) But it is designed for databases. The best of all, it's manufacturer-independent (Oracle, Microsoft, Teradata, etc) you can manipulate data in exactly the same way to:

- Data view (Select)  
- Insert  
- Update 
- Delete
